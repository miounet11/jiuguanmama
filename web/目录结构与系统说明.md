# 功能模块说明

- 玩法市场：展示、搜索、筛选、付费/免费标签、详情页
- 管理后台：玩法增删改查、图片上传、规则编辑、分类管理、审核发布
- 用户系统：注册、登录、权限分级（普通用户/管理员/审核员）
- 评论区：玩法详情页下方，支持用户评论与评分
- 审核流程：玩法提交、管理员审核、上线
- API 设计：RESTful 风格，支持玩法、用户、评论等数据交互
- SDK 接入：供第三方平台或游戏客户端快速集成玩法数据
