简介

欢迎来到九馆爸爸的玩法中心。这是一个面向玩家与运营者的集中展示页面，用于介绍、体验与管理不同的玩法模式。网站采用静态前端，便于部署到任何静态托管服务（GitHub Pages、Netlify、Vercel 等）。

功能概览

  • 玩法模式分类与描述
  • 每个玩法的要点、规则与示例
  • 可导出的玩法数据（JSON 格式）用于后台接入
  • 简单的试玩交互（本地模拟）
  • 面向后续扩展的插件化支持说明

玩法以 JSON 格式描述，方便后端、SDK 或插件直接读取。请参考下方的示例与 `modes.json`。

  • name：玩法名称
    • 
        
  • category：玩法类别（对战、社交、解谜 等）
    • 
        
  • description：玩法简短介绍
    • 
        
  • rules：玩法规则要点（数组）
    •
